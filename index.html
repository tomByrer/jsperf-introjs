A bookmarklet to run a tutorial on jsPerf.com homepage using Intro.js.

If you don't know, jsPerf is a croud-sourced JavaScript speed test site; helpful if you want to benchmark one function against another. Disclaimer: This project is not connected to jsPerf or maintainers.

jsPerf-tutorial Usage

  1. Add bookmarklet to your bookmarks toolbar:
    • Drag this link: [ jsPerf-tutorial ], then release on your bookmarks toolbar.
    • Or edit a bookmark by hand, copy-pasting this into the location field:
      javascript:(function(){var%20e=document.createElement("SCRIPT");e.type="text/javascript";e.src="//tombyrer.github.io/jsperf-tutorial/jsperf-tutorial.js";document.getElementsByTagName("body")[0].appendChild(e)})();
      (For Android, you might have to swap %20 for an actual space.)
  2. Go to add test | http://jsperf.com/, or enter /edit after an existing test.
  3. Run bookmarklet from your toolbar.
  4. After a few seconds, at the upper-right hand corner of the page, you'll see a "Run tutorial." banner. Click there to run the tutorial. Use the [Enter] key or the [Next ->] button to step though the tutorial.
